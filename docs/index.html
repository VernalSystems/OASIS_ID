The basics of a OASIS ID

An OASIS ID must contain a few things:.

  1. The identifier
  2. The resolvers that are aware of the identifier
  3. The end points that can be called upon

The absolute minimum of a OASIS ID looks like this:

identifier ~ resolver . end point
